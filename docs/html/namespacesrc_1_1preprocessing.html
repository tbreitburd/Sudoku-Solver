◆ load_sudoku()

src.preprocessing.load_sudoku (   path)

Load the sudoku .txt file into a useable Numpy array.

This function takes in a path to a sudoku .txt file, reads the file into a series of strings for each row, checks that the file has the correct format, then picks out the numbers from the strings, dropping the separators, and returns a 9x9 numpy array of the sudoku.

Parameters
pathPath to the sudoku txt file
Returns
A 9x9 numpy array containing the sudoku numbers
Exceptions
FileNotFoundErrorRaised if the sudoku file is not found
ValueErrorRaised if the sudoku file has an incorrect format, this includes incorrect number of rows, columns, separators, and incorrect characters for the separators.
IOErrorRaised if the sudoku file cannot be opened
