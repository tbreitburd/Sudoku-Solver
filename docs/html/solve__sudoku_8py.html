Detailed Description

Sudoku solver using backtracking algorithm.

Using functions from modules preprocessing.py and solver_tools.py, it takes in a sudoku in txt format (with specific formatting) as input, as well as a backtracking type (forward, backward or ordered), and a boolean indicating whether to use only backtracking or not. The solver then solves it using a backtracking algorithm, and optionaly a candidate checking method. It then returns the solved sudoku in the same format as the input, printing it to the console and writing it to a txt file, in a directory called "sudoku_solutions".
