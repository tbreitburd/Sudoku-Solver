Backtracking algorithm to solve the sudoku, using a recursive function.

This function takes a sudoku in array form, the associated markup dataframe of possible values for the sudoku cells, a list of the cells that we need to backtrack through from that markup, the cell number we are currently looking at within that list, and returns a solved sudoku, if possible.

Parameters
sudokuA 9x9 numpy array containing the sudoku numbers
markup_A 9x9 dataframe containing lists of possible values for each cell.
backtrack_cellsA numpy array containing the indexes of the cells that still have more than one possible value, in that markup file.
cell_numThe index of the current cell we are looking at, within the backtrack_cells list.
Returns
A boolean, True if the sudoku is solved, False if it is still not solved.

Check if the sudoku is valid.

This function takes in a sudoku, and checks if it is valid, i.e. a maximum of one of each number in each row, column and box. Given the final_check boolean, when True, it also checks if the sudoku is solved, i.e. if there are no empty cells.

Parameters
sudokuA 9x9 numpy array containing the sudoku numbers
final_checkA boolean, True if it's to check the sudoku is solved
Returns
A boolean, True if the sudoku is valid, False if it is not. And a string, containing a message explaining why the sudoku is not valid.

src.solver_tools.markup (   sudoku)

Create a markup for the sudoku.

This function takes in a sudoku. And for each empty cell of that sudoku, lists the possible values for that cell, by checking the row, column, and box that cell is in for values that are already present. And it returns all these possible values in a dataframe, with corresponding row and column indexes.

Parameters
sudokuA 9x9 numpy array containing the sudoku numbers
Returns
A 9x9 dataframe containing lists of possible values for each cell.
