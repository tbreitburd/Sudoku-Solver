Solve the sudoku.

Solve the sudoku. This function takes in a sudoku, and solves it using the backtracking algorithm.

Parameters
sudokuA 9x9 numpy array containing the sudoku numbers
backtracking_typeA string, either "forward", "backward" or "ordered", specifying the type of backtracking to use.
Returns
None
Exceptions
RuntimeErrorif the sudoku is no longer valid after the markup and the backtracking algorithm is applied.
RuntimeErrorif the backtracking algorithm fails to solve the sudoku.
