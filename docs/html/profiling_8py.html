Detailed Description

This file contains the code to conduct profiling on the sudoku solver, and its different functions.

Using functions from modules preprocessing.py and solver_tools.py, it takes in the same arguments as when running the solver from the command line, the input file path, the backtracking type adn whether to only use backtracking or not. It then profiles the performance of the solver, using the cProfile module. It then prints the profiling results to the console, and writes them to a txt file in a directory called "profile". On can set it up to profile different functions, by commenting out the functions that are not to be profiled.
