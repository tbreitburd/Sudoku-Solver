solve_sudoku

Solve a sudoku using backtracking algorithm.

Solve a sudoku using backtracking algorithm. This function takes in a sudoku in txt format (with specific formatting) as input, as well as a backtracking type (forward, backward or ordered), and a boolean indicating whether to use only backtracking or not. The solver then solves it using a backtracking algorithm, and optionaly a candidate checking method. It then returns the solved sudoku in the same format as the input, printing it to the console and writing it to a txt file, in a directory called "sudoku_solutions".

Parameters
input_filePath to the sudoku txt file
backtracking_typeType of backtracking to use (forward, backward or ordered)
bactracking_onlyBoolean indicating whether to use only backtracking or not
Returns
Prints the solved sudoku to the console, and writes it to a txt file in a directory called "sudoku_solutions"
Exceptions
RuntimeErrorRaised if the sudoku is not valid at loading time, after markup, or after backtracking.
RuntimeErrorRaised if the backtracking algorithm fails to solve the sudoku.

To solve a sudoku in the file "sudoku.txt", using forward backtracking only, run the following command in the terminal, from the root of the repository:

1python -m src.solve_sudoku sudoku.txt 'forward' True
